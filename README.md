# TodoList Application

## Project Overview
This project is a simple command-line-based Todo List application developed for the Decentralized Technologies TD1. It allows users to manage their tasks effectively by adding, deleting, marking tasks as complete, and viewing their todo list.

---

## Features
- Add tasks to the todo list
- Delete tasks by number
- Mark tasks as completed
- Display tasks with clear status indicators
- User-friendly interface with a clear screen function for better visibility

---

## Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/tcboris/TodoList-PERIER_Charles-CODF2
   ```
2. Navigate to the project directory:
   ```bash
   cd https://github.com/tcboris/TodoList-PERIER_Charles-CODF2
   ```

--- 

## Usage
Run the application with:
    ```bash
    python todo_list.py
    ```

Follow the on-screen menu to manage your tasks:
- Add tasks by typing their descriptions.
- Mark tasks as complete or delete them by their task numbers.
- View your todo list in a clear and organized format.
